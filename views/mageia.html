Mageia

Mageia est un système d’exploitation libre, basé sur GNU/Linux. C’est un projet communautaire soutenu par une association.

Maegia est une distribution éditée par Maegia.org .

La première version de Maegia est sortie le 1er juin 2011.

Environnements de bureau : Plasma/Gnome/Xfce/LXDE/LXQT

Mageia sous l'environnement de bureau Plasma

Les plus

  • Maegia est disponible gratuitement.
  • Distribution stable, une version de Maegia ne sort que lorsqu'elle est finalisée.
  • Maegia est libre donc son code peut être réutilisé ou modifier librement et gratuitement.
  • Distribution française donc toute la documentation et le site support sont disponibles en francais.
  • Facilement accessible pour les personnes qui découvrent Linux.
  • La configuration de Maegia est simple.

Les Moins

  • Mageia a un peu de retard par rapport aux autres distributions Linux.
